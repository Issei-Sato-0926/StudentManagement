# StudentManagement

## 概要

受講生の個人情報や受講コース情報を管理するウェブアプリケーションです。

## 主な機能

- 受講生情報（受講コース含む）の管理（検索・登録・更新・削除）
- 各コースの申込状況管理

## 使用技術

| 種類      | 技術・ライブラリ    |
|---------|-------------|
| 言語      | Java        |
| フレームワーク | Spring Boot |
| DBアクセス  | MyBatis     |
| データベース  | MySQL       |
